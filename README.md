## About

This repository documents my self-learning journey in advanced Python.  
Each module represents concepts I studied, implemented, tested, and documented to strengthen my understanding of real-world Python development.

The focus is not just learning syntax, but understanding **how and when to apply Python concepts effectively.**

## Getting Started

### Clone Repository

```bash
git clone https://github.com/Monikashwari/Python-Advanced.git
cd Python-Advanced
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Examples

```bash
cd Decorators
python decorators_basics.py
```

### Run Tests

```bash
# Run all tests
pytest -v

# Run a specific test file
pytest "D://Python-Advanced/Pytest/test_calculator.py" 
```
